This project is dedicated to analyzing CSV files — a simple yet powerful tool designed to assist those who need to process and interpret data collected through research or observation. Whether you're working on a scientific study, conducting a survey, or organizing experimental results, this platform helps you turn raw numbers into meaningful insights.

We hope that this project will be useful to you :)

Instruction

How to use the project

Step 0 - Go to the Analysis page

Step 1 – Upload your CSV file To unlock the full functionality of the app, you must first upload a valid CSV file. Once the file passes validation, the other sections will become available.

Step 2 – Basic Statistics In this section, you can view key summary statistics and descriptive measures for your data.

Step 3 – Visualizations Generate insightful graphs and charts to explore patterns, trends, and distributions in your data. Visualizations can be downloaded for use in reports or presentations.

Step 4 – Predictive Modeling Use linear regression to build a model that predicts your target variable.

Once your file is uploaded, you can freely navigate between steps in any order — steps 2, 3, and 4 become independent after the file is loaded. Visualizations can be downloaded for use in reports or presentations.

Visualizations

Diagram types

Heatmap – This chart shows the correlation matrix between all numeric columns in your dataset. No column selection is required — it is automatically generated based on available numeric data.

Boxplot – To build this chart, you need to select two columns: one with categorical values and one with numeric values. It visualizes the distribution, central tendency, and potential outliers within each category.

Violin Plot – Similar to the boxplot, this chart also requires one categorical and one numeric column. It combines a boxplot and a density plot, providing a richer view of the data distribution.

Histogram – This visualization displays the frequency distribution of a single numeric column. It helps identify the shape of the data and detect skewness or patterns.

Scatter Plot – Requires two numeric columns. This chart is used to explore relationships or correlations between two continuous variables.
